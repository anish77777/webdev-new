How to Display Times and Dates in HTML

Use the <time> element to represent a specific moment in time. The datetime attribute is used to translate dates and times into a machine-readable format. This is important because it helps with search engine results and helps the browser process date and time information more effectively.

Example 1: Time Only

Here is an example using the time element to represent twenty hundred hours, or eight PM in the evening. The value for the datetime attribute must be a valid time string in 24-hour format.

The reservations are for 20:00 (which is 8:00 PM).

Example 2: Date and Time Together

Here is another example of using the time element to represent a particular date. The value for the datetime attribute is in the ISO 8601 format. The first part of that value is the year, month and day. The capital T in the value is a separator between the date and time. The fifteen hundred hours would be three PM in the afternoon.

The graduation will be on June 15 at 3:00 PM.

Example 3: Date Only

This example shows how to represent just a date without a specific time. The datetime attribute follows the year-month-day format.

Our company was founded on March 15, 2010.

Example 4: Event Schedule

Whenever you need to represent events, publication dates, or appointments, it is best to use the time element. Here's an example of an event schedule:

  • Registration opens: May 1 at 9:00 AM
  • Main event: May 2 at 2:30 PM
  • Closing ceremony: May 3 at 7:00 PM

Example 5: Publication Date

This article was published on January 15, 2024 at 8:30 AM and last updated on January 20, 2024.

Key Points to Remember:

  • Always use the datetime attribute for machine-readable format
  • The datetime value must follow ISO 8601 standard
  • Separate date and time with capital "T" (e.g., 2024-06-15T15:00)
  • Use 24-hour format for time in the datetime attribute
  • Display text (between the tags) can be in any user-friendly format
  • Helps search engines understand the actual date/time value

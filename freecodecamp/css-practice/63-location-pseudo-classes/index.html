Location Pseudo-classes

Target elements based on URL fragments and link status.

Understanding Navigation States

Location pseudo-classes like :link, :visited, and :target allow you to style elements based on the user's current destination or history within the document.

Link State Showcase

:link (Unvisited)

Styles links that have not been clicked yet.

New Frontier

:visited (Clicked)

Styles links the user has already explored.

Old Friend (Google)

:any-link

Matches any link regardless of visited status.

Summary

  • :link – Default state for unclicked links.
  • :visited – Feedback for explored destinations.
  • :any-link – Selects links in any state.
  • :target – Styles the active fragment identified in the URL.

Implicit vs Explicit

Intentional Layouts and Browser Automation

Defining Boundaries

Explicit Grid refers to the rows and columns you intentionally set up using grid-template-rows and grid-template-columns.

Implicit Grid are the tracks automatically created by the browser when you have more items than your explicit structure can hold.

1. Explicit Structure

Predefined

Explicit CSS

.container {
  display: grid;
  grid-template-columns: 100px 100px;
}

2. Implicit Adaptation

Auto-Generated

Automatic Overflow

.container {
  /* 2 defined columns */
  grid-template-columns: 100px 100px;
  /* Extra items create implicit rows */
  grid-auto-rows: 80px;
}

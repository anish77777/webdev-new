External Fonts

Leveraging Google Fonts and Font Squirrel

1. What is an External Font?

An external font is a file hosted on a separate server rather than being included in your project files. This gives you access to a vast library of typefaces without requiring users to have them installed locally.

2. Google Fonts

Google Fonts is the most popular service for free, high-quality web fonts. You can embed them using two primary methods:

Method A: <link> Element

Paste the snippet in your HTML <head>. Best for performance as it allows parallel downloading.

<link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
<link href="https://fonts.googleapis.com/css2?family=Roboto&display=swap" rel="stylesheet">

Method B: @import Rule

Paste at the very top of your CSS file. Convenient for organizing purely within stylesheets.

@import url('https://fonts.googleapis.com/css2?family=Roboto&display=swap');

4. Font Squirrel

Font Squirrel is a great alternative for downloading font kits to host locally or for using their "@font-face Generator" for specialized needs.

Key Step: Visit the "Webfont Kit" tab to check licenses and download a pre-configured CSS package.

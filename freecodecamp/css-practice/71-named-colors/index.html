Named Colors in CSS

Discover the 140 predefined color names used to quickly style web pages.

What Are Named Colors?

Named colors are predefined color names recognized by all modern browsers. They offer a simple, human-readable way to define colors without needing to remember hex codes or RGB values.

Example: Styling a Paragraph

p {
  color: red;
}

This text is styled with the "red" named color.

The Limitations

While convenient, named colors have a limited range of 140 options. They lack the precision required for complex designs or brand-specific shades that need exact hex or HSL values.

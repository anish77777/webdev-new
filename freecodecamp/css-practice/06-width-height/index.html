Width and Height

Controlling the physical dimensions of elements in the digital space.

The Basics

In CSS, the width and height properties are used to control the dimensions of elements on a webpage.

The width property specifies the horizontal dimension, while height specifies the vertical dimension.

Auto Value: If you do not specify a width or height, the default is set to auto. This lets the browser determine the size based on content, parent container, and display type.
100x100
.box {
  width: 100px;
  height: 100px;
  background-color: skyblue;
}

Units of Measurement

These properties can be defined in different units to achieve different behaviors:

  • Pixels (px): A fixed-size unit providing precise control.
  • Percentages (%): Relative to the parent container's size.
  • Viewport Units (vw, vh): Relative to the size of the browser window.

The "Minimum" Constraint

The min-width and min-height properties specify the absolute smallest an element can be.

Even if you set a smaller fixed width, the min- value will override it to ensure the element doesn't shrink too far.

Set to 50px,
Actual: 150x100
.box {
  width: 50px;
  min-width: 150px; /* Overrides width */
  height: 50px;
  min-height: 100px; /* Overrides height */
  background-color: lightcoral;
}

The "Maximum" Constraint

Conversely, max-width and max-height set a ceiling on how large an element can grow.

If the specified width/height is larger than the max- limit, the element will be restricted to the maximum value.

Set to 300x200,
Actual: 150x100
.box {
  width: 300px;
  max-width: 150px; /* Restricts width */
  height: 200px;
  max-height: 100px; /* Restricts height */
  background-color: lightgreen;
}

Priority and Precedence

CSS follows a specific order of operations when multiple dimension constraints are present:

Rule of Thumb: min-width/height has the highest priority, followed by the normal width/height, which is then capped by max-width/height.
  • Min Overrides Everything: If min-width is 500px and width is 300px, the result is 500px.
  • Max Limits Growth: If width is 1000px and max-width is 600px, the result is 600px.

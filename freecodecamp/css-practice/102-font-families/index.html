Font Families

Fallback Mechanisms and Priority

1. What is a Font Family?

A font family is a group of fonts that share a common design. All fonts in a family are based on the same core typeface but vary in weight, style, and width.

In CSS, we use the font-family property to define the design of our text.

p {
  font-family: Arial;
}

2. Priority & Fallback

You can specify multiple font families in order of priority. If the first font isn't found, the browser looks for the second, and so on.

h1 {
  font-family: Arial, Lato, sans-serif;
}
1
Arial (Primary choice)
2
Lato (Alternative option)
3
sans-serif (Browser default fallback)

Interesting Fact: browsers choose fonts character by character. If a font lacks a specific symbol (like an emoji), the browser will look for it in the fallback fonts!

3. Generic Font Families

Generic font families are defaults that browsers fall back to if your specific choices aren't available. 

.example {
  font-family: monospace;
}

List Style Properties

Control the appearance of your lists with list-style.

The list-style property is a shorthand for three properties that define how list markers behave and look. Understanding these allows for highly customized and professional list designs.

list-style: [type] [position] [image];
Type

list-style-type

Defines the type of marker (bullet or number). Standard values include disc, circle, square, decimal, and roman.

  • Square Marker
  • Customizable Shape
Position

list-style-position

Controls if the marker is inside or outside the list item's content box.

  • Inside: Marker aligns with the wrapped text lines.
Image

list-style-image

Replaces the default bullet with a custom image. Always provide a fallback list-style-type! 

ul {
  list-style-image: url('fcc-logo.png');
  list-style-position: inside;
}
  • Custom Branding
  • Unique Visual Style
  • Enhanced User Experience

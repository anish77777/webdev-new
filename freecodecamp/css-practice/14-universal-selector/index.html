The Universal Selector

Mastering the * selector and its absolute zero specificity.

What is it?

The universal selector (*) matches every single element in the document. It is the broadest selector available in CSS.

* {
  margin: 0;
  padding: 0;
}

This "Reset" technique is commonly used to remove default browser margins and padding, giving you a clean slate for your designs.

Specificity: (0, 0, 0, 0)

SCORE: 0.0.0.0

The universal selector has the lowest possible weight. It contributes nothing to the specificity calculation. This means literally any other selector will override its styles.

The Specificity Chain

Look at how different selectors target the same element. The one with the highest specificity (the ID) always wins the color battle.

/* Universal (Lowest) */
* { color: blue; }

/* Type Selector */
p { color: red; }

/* Class Selector */
.highlight { color: green; }

/* ID Selector (Highest) */
#unique { color: purple; }

This text is PURPLE because the ID wins!

Key Takeaways

• Use * for document-wide resets or broad defaults.

• It never "rolls over" or adds to specificity scores.

• It can be used globally or within a context (e.g., .container *).

CSS Custom Properties

Variables for Dynamic & Maintainable Design

Syntax & Declaration

Base

Custom properties (variables) allow you to store specific values and reuse them throughout your CSS. They start with two dashes --.

/* Declaration in :root */
:root {
  --main-color: #8b5cf6;
}

/* Usage with var() */
.button {
  background-color: var(--main-color);
}

Scoping & Cascading

Logic

Variables follow the cascade. You can define them globally in :root or locally within a specific class to override defaults.

Responsive Variables

Advanced

One of the most powerful uses of variables is combining them with media queries. Instead of changing multiple properties, you just update the variable value!

Responsive Card

Resize your browser to see the width, background, and padding update automatically via variables.

@media (min-width: 768px) {
  :root {
    --card-bg: #1e293b;
    --card-width: 70%;
  }
}

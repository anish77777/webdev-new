The fr Unit

Creating Truly Flexible and Responsive Layouts

Fractional Proportions

While percentages can create responsive grids, the fr unit is a game-changer. It represents a fraction of the available space in the container.

Think of it as slicing a pie – if you have 4 slices (1fr each), everyone gets an equal piece, no matter how big the pie is.

1. The Percentage Approach

Fixed Ratio

Percentage CSS

.container {
  display: grid;
  grid-template-columns: 25% 25% 25% 25%;
  gap: 15px;
}

Current Layout

2. The fr Unit Approach

Proportional

Fractional CSS

.container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr 1fr;
  gap: 15px;
}

Dynamic Layout

Pro Tip: Unlike percentages, fr units account for the gap before distributing space. This prevents layout overflow and calculation headaches!

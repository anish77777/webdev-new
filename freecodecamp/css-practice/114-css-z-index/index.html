The Z-Index

Controlling Vertical Stacking & Layering

01. Stacking Order

Vertical Axis

The z-index property controls which elements appear "closer" to the viewer. When positioned elements overlap, the one with the higher z-index value will be placed on top.

Visual Stacking Demo

z-index: 1
z-index: 3
z-index: 2

Even though "z-index: 2" comes last in the HTML, it's underneath "z-index: 3" because 3 is a higher layer.

.top-layer {
  position: absolute;
  z-index: 10; /* Higher = Closer */
}

02. The "Positioned" Rule

Requirement

CRITICAL: z-index only works on elements which have a position value other than static (the default). If your element is static, z-index will be ignored.

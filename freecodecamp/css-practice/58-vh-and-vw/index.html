VH & VW Units in CSS

Master viewport dimensions. Learn how to size elements relative to the browser window for seamless full-screen experiences and fluid typography.

üìè Understanding vh & vw

These are viewport-relative units. vh stands for "viewport height" and vw stands for "viewport width". Each unit represents 1% of the respective browser dimension.

1vh
1% of Viewport Height
1vw
1% of Viewport Width

üöÄ Full-Screen Hero Sections

Viewport units are essential for hero sections that need to cover the entire screen. Setting height: 100vh ensures the section fills the viewport depth regardless of the physical pixel count.

Hero Layout

Sized with 100vh / 100vw

This layout will adapt its size in real-time as the browser window resizes.

Full Screen CSS

.hero {
width: 100vw;
height: 100vh;
background: url('hero-bg.jpg');
}

üî† Fluid Typography

By using vw units for font-size, text will smoothly scale up and down with the width of the browser window. This creates a "breathable" design on any device.

Responsive Title

Resize your screen to see this header scale!

Viewport Scaling Fonts

h1 {
font-size: 5vw;
}

What Is Overflow?

Overflow occurs when content (like text or images) exceeds the size of its containing element. It's fundamentally two-dimensional:

Horizontal Overflow

Y-Axis

Vertical Overflow

Option 1: overflow-y: hidden

This property hides any content that extends beyond the vertical height of the container. While it fixes layout issues, the hidden content becomes unreachable.

Pro Tip: auto

The auto value lets the browser decide. It will only show scrollbars if the content actually overflows, keeping your UI clean.

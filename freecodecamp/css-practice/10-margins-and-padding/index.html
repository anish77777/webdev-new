Spacing Fundamentals

Margin and padding are essential properties in CSS for creating well-structured, readable, and visually appealing web pages.

Margins control the space outside an element, helping to separate it from other elements and define the layout structure.

Padding controls the space inside an element, improving content readability and aesthetic appeal.

Single Side Spacing

To apply spacing to a specific side, you can use property-specific names like margin-top.

<p>Paragraph one</p>

Paragraph one (20px top margin)

<p>Paragraph two</p>

Paragraph two (20px top margin)

p {
  margin-top: 20px;
  border: 2px solid black;
}

In this example, we apply 20px of margin only to the top of each paragraph element.

The Power of Shorthand

Specify one, two, three, or four values at once using the margin shorthand property.

One Value (All sides)

10px all sides

Two Values (Top/Bottom, Left/Right)

Paragraph one

10px Vertical, 20px Horizontal

p { margin: 10px 20px; }

Three Values (Top, Left/Right, Bottom)

Paragraph one

10px Top, 20px L/R, 30px Bottom

p { margin: 10px 20px 30px; }

Four Values (Top, Right, Bottom, Left)

Paragraph one

T:10, R:20, B:30, L:40

p { margin: 10px 20px 30px 40px; }

CSS Padding

The padding property is used to apply space inside the element, between the content and its border.

Paragraph one

Internal Padding Applied

Paragraph three 
p {
  padding: 10px 20px 30px 40px;
  border: 2px solid black;
}

As you can see, padding is applied to the content inside the border, unlike margin which applies to the outside.

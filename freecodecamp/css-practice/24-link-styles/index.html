Link Styles & Usability

Mastering default link behaviors for intuitive navigation.

Default link styles play a crucial role in enhancing usability and accessibility. Standard colors like blue for unvisited and purple for visited links provide immediate visual cues that users rely on.

Standard

Default Behaviors

Browsers use specific colors and underlines to distinguish links from regular text.

Check out these unvisited links and visited links.

UX

Visual Cues

Underlines provide redundant cues for colorblind users, ensuring they can still identify interactive elements.

a {
  text-decoration: underline;
}
Modern

Customizing Link Styles

While maintaining standard colors is important, you can modernize the look by replacing the default underline with a border-bottom.

a {
  text-decoration: none;
  border-bottom: 1px solid blue;
}

a:visited {
  border-bottom: 1px solid purple;
}

Modern CSS

Underline Mastery

Modern CSS provides properties to control the underline without resorting to borders. This keeps the decoration semantically linked to the text.

text-decoration-color

Independent underline color

Colorful Underline
text-decoration-thickness

Underline weight

Bold Underline
text-underline-offset

Spacing from text

Offset Underline 
a {
  text-decoration-color: #facc15;
  text-decoration-thickness: 3px;
  text-underline-offset: 5px;
}

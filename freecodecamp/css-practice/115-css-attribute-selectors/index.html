Attribute Selectors

Targeting Elements by Their Properties

01. Existence & Equality

Theory

Attribute selectors target elements based on the presence or value of an attribute. They are powerful for styling dynamic content or form elements without adding extra classes.

Visual Demo: Links

Has Href No Href Has Title 
/* Target any element with an href attribute */
a[href] {
  color: var(--accent-primary);
}

/* Target elements with both href AND title */
a[href][title] {
  border-bottom: 2px solid var(--accent-secondary);
}

02. Pattern Matching

Advanced

You can target elements based on substrings within an attribute's value using special operators.

[attr^="val"] Starts with "val" https://secure.com
[attr$="val"] Ends with "val" image.jpg
[attr*="val"] Contains "val" my-cool-site 
/* Starts with "https" */
a[href^="https"] { color: green; }

/* Ends with ".pdf" */
a[href$=".pdf"] { background: red; }

03. Language & Data

Context

Use the lang attribute for accessibility and localization, and data-* attributes for custom application state.

Multilingual Demo

Hello World (en)

Bonjour le monde (fr)

Hola Mundo (data-lang="es")
/* Target standard HTML lang attribute */
p[lang="fr"] {
  font-style: italic;
  border-left: 3px solid var(--accent-tertiary);
}

/* Target custom data attribute */
div[data-lang="es"] {
  color: var(--accent-primary);
}

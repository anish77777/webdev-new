Best Practices for Images

Adding images to your websites is a great way to engage your users and increase visual appeal, but it requires mindful design and optimization.

1. Responsive Images

Responsive images are images that scale to fit the size of the screen they are being viewed on. This ensures your images look good on all devices, from desktops to mobile phones.

img { max-width: 100%; height: auto; }

Resize your screen to see this image scale automatically!

2. Resolution & PPI

Pixels per inch (PPI) is the number of pixels in one inch of an image. Higher PPI means better quality. High-quality images look professional, but for the web, they must be properly optimized to ensure fast loading.

Always use high-resolution images that are optimized for web use to prevent slowing down your site.

3. Balance & Hierarchy

Balance is the distribution of visual weight. Pair text blocks with images evenly to create a harmonious design.

Hierarchy is the order in which elements are viewed. Place important images higher to connect users emotionally with your mission right away.

4. Alignment

Alignment is the placement of elements in relation to each other. Aligning images with text creates a cohesive layout where elements feel like they belong together. This improves readability and reinforces structure.

.aligned-content { display: flex; align-items: center; gap: 20px; }

5. Accessibility

Ensuring images are accessible means using alt text. Screen readers read this text to users with visual impairments, allowing everyone to understand your content.

<img alt="Descriptive text here" >

What Are Hex Codes?

Precision color selection using hexadecimal notation

The Hexadecimal Model

A hex code is a concise way to represent colors in the RGB color model. It uses the base-16 numbering system (0-9 and A-F) to specify the intensity of red, green, and blue.

# RR GG BB

Shorthand Notation

When both characters in each pair are the same, you can use a 3-digit shorthand. For example, #FFFFFF becomes #FFF.

Full Code Shorthand Color
#FFFFFF #FFF White
#000000 #000 Black
#007BFF - Azure

CSS Basics

Cascading Style Sheets (CSS) is a markup language used to apply styles to HTML elements. It controls colors, layouts, backgrounds, and more.

Anatomy of a Rule

CSS Syntax
selector {
  property: value;
}

Viewport Meta

The viewport meta tag instructs the browser on how to scale the page on mobile and tablet devices.

Browser Defaults

User-Agent styles are the default settings (like margins on <p> tags) applied by the browser natively.

Application Methods

Inline

Applied directly via the style attribute. High priority, but hard to maintain.

Internal

Written inside <style> tags in the HTML <head>.

External

Linked via <link> in a separate file. Best Practice for large projects.

Margin, Padding & Size

The CSS Box Model controls how much space an element takes up and how its content is spaced.

Width & Height

Supports min and max constraints. Default is auto.

Margin

Space outside the element border.

Padding

Space inside the element, between content and border.

Shorthand Notation

margin: 10px; /* All 4 sides */
margin: 10px 20px; /* Top/Bottom, Left/Right */
margin: 10px 20px 30px 40px; /* Top, Right, Bottom, Left */

CSS Combinators

Descendant

ul li targets all LI items inside a UL.

Child (>)

.parent > p targets only direct children.

Next Sibling (+)

Targets the element immediately following another.

Subsequent (~)

Targets all siblings that come after an element.

Visual Simulation
I am a direct child (Styled by >)
I am nested (Not styled by >)
Reference Point
Next Sibling (Styled by +)
Later Sibling (Styled by ~)

Display Levels

Inline

Only takes up needed width. No new line. e.g., <span>, <a>.

Block

Starts on a new line, takes full width. e.g., <div>, <p>.

Inline-Block

Flows like inline, but accepts width and height settings.

Specificity Hierarchy

Specificity is a score (a, b, c, d) that determines which rule wins during a conflict.

Selector Type Score Example
!important Max color: red !important;
Inline Style (1, 0, 0, 0) style="color: red"
ID Selector (0, 1, 0, 0) #main-header
Class / Attr (0, 0, 1, 0) .nav-item
Type / Tag (0, 0, 0, 1) h1, p
Universal (*) (0, 0, 0, 0) *

Cascade & Inheritance

Cascade Algorithm

Filters by Relevance → Origin → Specificity → Order of Appearance.

Inheritance

Traits (like fonts/color) passing from Parent to Child. Use inherit to force it.

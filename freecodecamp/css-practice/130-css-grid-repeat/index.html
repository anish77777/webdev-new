Repeating Tracks

Concise Layout Definitions with repeat()

The Power of Conciseness

Manually typing 1fr 1fr 1fr 1fr works, but it's redundant. The repeat() function allows you to define a section or the entirety of your track listings efficiently.

It takes two main arguments: the repeat count and the tracks you wish to repeat.

1. Simple Repetition

Clean Code

Standard Refactor

.container {
  display: grid;
  grid-template-columns: repeat(4, 1fr);
  column-gap: 10px;
}

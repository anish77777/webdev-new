Vivid Gradients

A comprehensive guide to CSS transitions. Learn every syntax, every value, and master the art of color.

üìè Linear Gradients

Transitions colors along a straight line. You can define the direction and the color points (stops) involved.

Syntax

background: linear-gradient(direction, color-stop1, color-stop2, ...);

Possible Directions

Keywords

to top, to bottom, to left, to right, to top right, etc.

Angles

45deg, 90deg, 180deg, -90deg, etc.

linear-gradient(to right, red, yellow)

Color Stops

Specify the colors and positions where the gradient transitions occur. Positions can be in % or px.

linear-gradient(45deg, cyan 10%, purple 50%, red 90%)

üéØ Radial Gradients

Transitions radiating from an origin outward in circular or elliptical shapes.

Syntax

background: radial-gradient(shape size at position, color-stop1, color-stop2, ...);

Shapes

circle, ellipse

Sizes

closest-side, closest-corner, farthest-side, farthest-corner, contain, cover

Positions

center, top left, 50% 50%, 10px 20px

Advanced Example

Circular gradient centered, starting red, transitions to yellow at 50%, and ends green.

radial-gradient(circle closest-side at center, red, yellow 50%, green)

Elliptical gradient at top left:

radial-gradient(ellipse farthest-corner at top left, indigo, purple)

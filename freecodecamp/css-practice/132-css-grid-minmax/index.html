The minmax()

Defining Fluid Boundaries for Grid Tracks

Dynamic Range Control

The minmax() function defines a range for the size of a grid track. It ensures a track never shrinks below a minimum or grows above a maximum size.

Syntax: minmax(min, max)

1. Practical minmax Example

Boundary Control

Fluid CSS

.container {
  display: grid;
  grid-template-columns: 
    minmax(150px, 300px) 1fr;
  gap: 10px;
}

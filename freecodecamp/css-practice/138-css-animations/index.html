CSS Animations

Bringing Your Web Elements to Life

Dynamic Motion

CSS animations allow you to create dynamic, visually engaging effects without JavaScript. They smoothly transition elements between styles over a specified duration.

At its core, a CSS animation consists of two main components: the @keyframes rule and the animation property.

1. The @keyframes Rule

Defining Stages

The @keyframes rule defines the stages and styles of the animation. Percentages represent progress, with 0% being the start and 100% the end.

@keyframes slide-in {
  0% { transform: translateX(-100%); }
  100% { transform: translateX(0); }
}
Hello, I slide in!

2. Color Cycle Demo

Infinite Loop

You can define multiple stages for complex transitions. Here, we cycle background colors across red, blue, and green.

.complex-animation {
  animation: color-change 3s linear infinite alternate;
}
Watch my colors change!

3. Project: The Ferris Wheel

Complex Motion

Combining multiple animations, rotation origins, and timing functions allows for complex mechanical simulations like this Ferris Wheel.

The wheel rotates clockwise, while the cabins rotate counter-clockwise to stay upright.

.wheel { animation: wheel 10s linear infinite; }
.cabin { animation: cabins 10s ease-in-out infinite; }

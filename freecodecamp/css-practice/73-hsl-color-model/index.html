What Is the HSL Color Model?

Hue, Saturation, and Lightness: The human-centric color model

One of the most versatile and intuitive ways to work with color is by using the HSL color model. HSL stands for Hue, Saturation, and Lightness — three key components that define a color in a way that aligns with how humans perceive it.

Hue

The color type, represented as an angle on the color wheel (0° to 360°). 0° is red, 120° is green, and 240° is blue.

Saturation

The intensity or purity of the color (0% to 100%). 100% is fully vivid, while 0% appears as grayscale.

Lightness

How light or dark the color is (0% to 100%). 0% is black, 50% is the normal tone, and 100% is white.

Transparency with HSLA

Just like RGB has RGBA, HSL has HSLA. The fourth parameter is the alpha value, controlling opacity from 0 (transparent) to 1 (opaque).

Semi-transparent Red (0.5 Alpha)
background-color: hsla(0, 100%, 50%, 0.5);

Targeting Ordered Lists

Styling Based on Numbering Type

01. The Type Attribute

Concept

The type attribute on an <ol> element defines the kind of marker to use (numbers, letters, or numerals). We can target these specific lists using attribute selectors.

Default / Decimal (type="1")

  1. First Item
  2. Second Item
  3. Third Item
/* Target standard numbered lists */
ol[type="1"] {
  color: var(--text-main);
}

02. Alphabetical Lists

A vs a

You can distinguish between uppercase (A) and lowercase (a) alphabetical lists.

Uppercase (type="A")

  1. Phase One
  2. Phase Two
  3. Phase Three

Lowercase (type="a")

  1. sub-point one
  2. sub-point two
  3. sub-point three
/* Uppercase Letters */
ol[type="A"] {
  color: var(--accent-primary); /* Purple */
  font-weight: bold;
}

/* Lowercase Letters */
ol[type="a"] {
  color: var(--text-muted);
}

03. Roman Numerals

Classic

Similarly, Roman numerals (I, i) can be styled distinctly.

Uppercase (type="I")

  1. Introduction
  2. Methodology
  3. Conclusion

Lowercase (type="i")

  1. preface
  2. acknowledgments
  3. dedication
/* Lowercase Roman Numerals */
ol[type="i"] {
  color: var(--accent-secondary); /* Green */
  font-style: italic;
}

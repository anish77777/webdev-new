The box-sizing Property

This property controls how the width and height of elements are calculated. By default, browsers use content-box, but border-box is often preferred for responsive layouts.

* { box-sizing: border-box; }

1. content-box (Default)

Width and height only apply to the content area. Padding and borders are added on top of the specified size.

Total Width = Width + Left/Right Padding + Left/Right Border

2. border-box

Width and height include content, padding, and border. The element's total size stays fixed.

Specified Size = Total Rendered Size

Actual Render: 300px x 200px

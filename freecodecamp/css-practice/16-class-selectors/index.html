Class Selectors

Versatile and efficient targeting for modern CSS architecture.

What are Class Selectors?

Class selectors are defined by a period (.) followed by the class name. They allow you to apply the same styling to multiple elements across your document, making them one of the most powerful tools for consistent design.

CSS
.highlight {
  color: green;
}

I am a paragraph with the "highlight" class. I am colored green!

Specificity: (0, 0, 1, 0)

SCORE: 0.0.1.0

Class selectors have a **moderate** level of specificity. They sit comfortably in the third column of the calculation, meaning they will always override tag names (Type Selectors) but will be gracefully overridden by IDs or Inline styles.

Combined Selectors

Class selectors can be combined with other selectors to create highly specific rules. For example, you can target only paragraph elements that have a specific class.

Qualified Selector
p.bold-text {
  font-weight: bold;
}

I am a bold paragraph because I match both the tag (p) and the class (.bold-text).

I am a div with the same class, but I am not bold because the rule only targets 'p' tags.

Summary

• Use classes for reusable styles and components.

• Classes are better for maintainability than IDs for general styling.

• Combining tag and class (e.g., p.class) increases specificity to (0, 0, 1, 1).

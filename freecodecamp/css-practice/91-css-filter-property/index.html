What is the Filter Property?

The filter property allows you to apply graphical effects like blurring, color shifting, and brightness adjustments to elements without modifying the original asset.

selector { filter: function(amount); }

Combining Filters

One of the most powerful aspects of filters is that you can combine multiple effects by separating them with spaces.

img { filter: contrast(150%) brightness(110%) sepia(30%); }

Styling Lists

line-height Property

Used to create space between lines of text. Values include normal, numbers, percentages, and length units like em.

line-height: 1.6;

list-style-type

Specifies the marker for a list item (e.g., circle, disc, decimal, square).

  • Item 1
  • Item 2
  • Item 3
ul { list-style-type: square; }

list-style-position

Sets the marker position: inside or outside.

  • Inside position item
  • Outside position item

Spacing with Margins

Use margin-bottom on li elements to control the gap between items.

.list li { margin-bottom: 20px; }

Styling Links & Pseudo-classes

Pseudo-classes select elements based on state:

:link - Unvisited
:visited - Visited
:hover - Active hover
:focus - Received focus
:active - Activated (click)
a:hover { color: green; text-decoration: underline; }

Working with Backgrounds

background-size

cover fills the element; contain fits within it.

background-repeat

Default is repeat. Use no-repeat to stop tiling.

background-attachment

scroll (default) or fixed (parallax effect).

The Shorthand Property

background: center top no-repeat url("image.jpg");

Media Breakpoints

Defining the Turning Points of Design

The Strategy

Theory

Breakpoints are specific viewport widths where a website's layout and content adjust to accommodate different screens. They are the "switch" points in your media queries.

576px

Small Devices (Smartphones)

768px

Medium Devices (Tablets)

992px

Large Devices (Desktops)

1200px

Extra Large (Wide Monitors)

/* Base: Small Mobile */
.adaptive-grid { grid-template-columns: 1fr; }

@media (min-width: 576px) {
  /* Small Devices */
  .adaptive-grid { grid-template-columns: 1fr 1fr; }
}

@media (min-width: 992px) {
  /* Large Devices */
  .adaptive-grid { grid-template-columns: repeat(4, 1fr); }
}

Fixed & Sticky

Controlled Scrolling & Persistent Elements

01. Fixed Positioning

Viewport

position: fixed removes an element from the flow and fixes it relative to the viewport. It stays in the exact same spot regardless of scrolling. This is perfect for navigation bars or "Back to Top" buttons.

Fixed Header (I never move)

Scroll down inside here...

Look up! The header is still there.

header {
  position: fixed;
  top: 0;
  width: 100%;
}

02. Sticky Positioning

Hybrid

position: sticky is a hybrid. It behaves like relative (staying in the flow) until the user scrolls it past a specific threshold (like top: 10px), then it "sticks" like fixed.

Scroll slowly...

I'm Sticky!

I caught you!

h2 {
  position: sticky;
  top: 0;
}

The Big Three

Type Relative To In Flow? Scroll Behavior
Absolute Nearest Positioned Ancestor No Scrolls with parent
Fixed Viewport No Stay in place
Sticky Nearest Scroll Parent Yes Sticks at threshold

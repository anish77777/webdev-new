Grid Placement

Positioning Items Identically via Grid Lines

The Line System

Grid items are placed using grid lines. These are the numbered lines that separate your rows and columns. Placing an item involves specifying its starting line and ending line.

Syntax: grid-column: start / end; or 1 / span 2;

1. Spanning Columns & Rows

Structural Control

Placement CSS

.featured-item {
  grid-column: 1 / span 2;
  grid-row: 1 / 3;
}

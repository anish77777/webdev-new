The CSS @property Rule

Type-Safe and Animatable Custom Properties

Syntax & Control

Advanced

The @property rule gives you granular control over custom properties, defining their type, inheritance, and initial values.

@property --main-color {
  syntax: '<color>';
  inherits: false;
  initial-value: #3498db;
}

What is a CSS Reset?

A CSS reset is a stylesheet that removes default browser formatting, providing a consistent baseline for your custom styles. Different browsers (Chrome, Firefox, Edge) apply different default margins, paddings, and styles to HTML elements.

Why use it? It minimizes inconsistencies and makes the styling process easier by giving you a "blank canvas."

1. Custom Resets

You can create your own reset rules for precise control. A common starting point is resetting margins and padding for all elements:

* {
  margin: 0;
  padding: 0;
  box-sizing: border-box;
}

The * wildcard selector matches everything, setting a uniform starting point.

2. Third-Party Resets

For more robust solutions, many developers use battle-tested libraries:

Normalize.css

Makes browsers render all elements more consistently and in line with modern standards. It preserves useful defaults.

sanitize.css

Ensures default styles are consistent across modern browsers, correcting common bugs and inconsistencies.

Accessibility Matters

When resetting styles, be careful not to remove properties that are crucial for accessibility, such as focus outlines for keyboard users.

Never use outline: none; without providing an alternative focus state!

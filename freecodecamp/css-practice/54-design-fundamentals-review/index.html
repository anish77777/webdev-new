Layout

How visual elements are arranged on a page to communicate a message.

Hierarchy

Establishes the order of importance, ensuring key info is noticed first.

Contrast

Creating clear distinctions via color, size, or shape to improve readability.

Balance

Distribution of visual weight via symmetrical or asymmetrical arrangements.

White Space

The empty area surrounding elements, also known as negative space.

UX

How users feel: intuitive, easy to use, efficient, and enjoyable.

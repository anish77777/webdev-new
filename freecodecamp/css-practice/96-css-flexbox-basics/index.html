1. What Is CSS Flexbox?

CSS flexbox is a one-dimensional layout model that allows you to arrange elements in rows and columns within a container. It focuses on arranging elements along a single axis at a time (horizontal or vertical).

When to Use: Use it to create responsive websites that adapt to different screen sizes and orientations.

2. Container & Items

A Flex Container is created with display: flex. Its direct children become Flex Items.

Default Behavior (Block)

Without Flexbox, div elements stack vertically.

Flex Behavior

With display: flex, items align horizontally and shrink to fit.

3. The Flex Model: Axes

Every flex container has two perpendicular axes:

  • Main Axis: Primary direction (Horizontal by default).
  • Cross Axis: Perpendicular to main axis (Vertical by default).

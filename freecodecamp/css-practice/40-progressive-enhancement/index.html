Progressive Enhancement

A design approach that ensures essential content is accessible to all users, regardless of browser or connection power.

Evolving the Experience

Progressive enhancement focus on delivering a core experience that works for everyone, while offering extra features and improvements to users with more advanced browsers or better internet connections.

The Core Principles

  • All core content and basic functionality should be accessible on all browsers
  • All advanced layouts should be provided through external CSS stylesheets
  • All advanced functionality should be provided through external JavaScript files
  • A user's browser preferences should be respected

Speed & Accessibility

Using this approach makes applications more accessible because core content is never blocked. For users on slow connections, the browser downloads necessary resources first, ensuring they can read the content even if the "bells and whistles" take longer.

SEO & Crawlability

Search engines can easily crawl your application because the core content is available in the initial HTML response. This improves visibility compared to apps that require complex JS execution just to show basic text.

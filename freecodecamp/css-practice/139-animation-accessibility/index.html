Inclusive Motion

Ensuring Animation Accessibility for Everyone

Why Accessibility Matters

Animations can enhance UX, but they can also cause dizziness, nausea, or even seizures for users with vestibular disorders or photosensitivity. Motion sensitivity is a physical reality that developers must respect.

1. prefers-reduced-motion

Media Query Control

The prefers-reduced-motion media query detects if a user has requested minimal animations at the system level.

Using a "blanket override" ensures that for those users, animations are almost instantaneous and non-distracting.

@media (prefers-reduced-motion: reduce) {
  * {
    animation-duration: 0.01ms !important;
    scroll-behavior: auto !important;
  }
}

2. Targeted Overrides

Precise Logic

Instead of a blanket ban, you can provide alternative, less motion-intensive experiences for specific elements like scrolling banners or pulsing buttons.

CSS Pseudo-classes

Select elements based on their state or position.

What are Pseudo-classes?

Pseudo-classes are special keywords added to selectors that specify a special state of the selected elements. They let you style elements dynamically based on user interaction or their position in the document.

Syntax
selector:pseudo-class {
  property: value;
}

User Interaction States

These pseudo-classes react to how a user interacts with elements like buttons and links.

:hover

Triggered when the mouse pointer is over an element.

Hover over me!

:active

Triggered while an element is being activated (clicked).

:disabled

Styles an element that is in a disabled state.

:visited

Styles a link that the user has already visited.

Summary

  • :hover – User mouses over element.
  • :active – Element is being clicked.
  • :disabled – Element cannot be interacted with.
  • :first-child / :last-child – Target based on parent order.
  • :visited – Target previously clicked links.

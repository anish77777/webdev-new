Breadcrumbs Best Practices

Guide your users through complex hierarchies with clear, intuitive navigational aids.

What are Breadcrumbs?

Breadcrumbs are a navigation aid that shows the user where they are in the site's hierarchy. They help users understand their location and provide an easy way to move back to previous levels.

Home / Electronics / Phones / Smartphone XYZ
/* HTML Structure */ <nav aria-label="Breadcrumb"> <ol> <li><a href="/">Home</a></li> <li>Smartphone XYZ</li> </ol> </nav>

1. Choosing a Separator

The separator is the character that separates different levels of the hierarchy. Common choices include `>`, `/`, or `Â»`.

Home > Electronics > Smartphone XYZ
.breadcrumb li + li::before { content: ">"; padding: 0 8px; }

2. Placement

Breadcrumbs are typically placed at the top of the page, either above or below the main navigation bar. They should be visible and easy to find without struggling.

Home Products About
Home > Products

3. Sizing & Hierarchy

Breadcrumbs should be large enough to read but secondary to primary navigation. They are most useful for sites with deep hierarchies, like e-commerce or technical documentation.

.breadcrumbs { font-size: 0.9rem; color: var(--text-muted); }

What is RGB?

RGB stands for Red, Green, and Blue â€” the primary colors of light. The RGB color model is an additive color model, meaning colors are created by combining light at varying intensities.

Intensity ranges from 0 (no light) to 255 (full light).

  • Black: rgb(0, 0, 0)
  • White: rgb(255, 255, 255)
  • Red: rgb(255, 0, 0)

Transparency with RGBA

The rgba() function adds an alpha value to control transparency, ranging from 0 (transparent) to 1 (opaque).

background-color: rgba(0, 0, 255, 0.5);

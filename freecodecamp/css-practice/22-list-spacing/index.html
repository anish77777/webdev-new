How Do You Space List Items?

Mastering margin and line-height for beautiful lists.

Margins and line-height are essential for spacing list items to enhance readability and visual appeal.

Method 1

Spacing with Margins

Margins create space outside each list item. This method is perfect for controlling the gap between separate items.

li {
  margin-bottom: 40px;
}

Method 2

Spacing with Line-Height

The line-height property adjusts the vertical spacing within a single list item.

li {
  line-height: 2;
}

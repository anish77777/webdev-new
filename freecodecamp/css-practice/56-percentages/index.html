Percentages in CSS

Master proportions. Learn how to use relative percentage units to create fluid, adaptive, and perfectly balanced layouts.

üîó Relative Proportions

Percentages are relative units that define dimensions as a proportion of their parent element. Setting width: 50% tells the element to occupy exactly half of its container.

Interact: Adjust Width
50%

Fluid Child CSS

.container { width: 400px; }

.box {
width: 50%;
height: 100%;
background-color: red;
}

Using percentages for images ensures they scale down gracefully on smaller screens. max-width: 100% is a standard practice for responsive web design.

Try resizing your browser to see this image scale proportionally!

Responsive Image CSS

img {
max-width: 100%;
height: auto;
}

üéØ Vertical Centering

Percentages are handy for absolute positioning. Setting top: 50% moves the top edge to the middle, and translateY(-50%) offsets the element by half its own height to perfectly center it.

CENTERED

Perfect Centering CSS

.centered {
position: absolute;
top: 50%;
left: 50%;
transform: translate(-50%, -50%);
}

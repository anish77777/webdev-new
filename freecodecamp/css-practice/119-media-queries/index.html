Media Queries

Conditional Styling for Modern Web

Syntax & Types

Foundation

A media query checks for specific conditions (like viewport width or device type) and applies CSS only if they are true.

Syntax: @media type and (feature: value) { ... }

Media Types

  • screen: Primary monitor/mobile display.
  • print: Paper or print-preview.
  • all: Default category for all devices.

Modern Features

  • min-width: Range-based checks.
  • orientation: landscape or portrait.
  • hover: Checks for mouse support.

Mobile-First Enhancement

Interactive

Try resizing your browser. The box below will "Level Up" as the screen gets wider!

Mobile View

This is the base style (no media query).

/* Base styles (Mobile) */
.breakpoint-box { background: #111; }

/* Tablet (+) */
@media screen and (min-width: 768px) {
  .breakpoint-box { 
     background: #0ea5e9; 
     transform: scale(1.1);
  }
}

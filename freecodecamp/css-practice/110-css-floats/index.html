CSS Floats Masterclass

Understanding the Flow and the Clearfix Technique

What Are Floats?

Floats in CSS are a technique originally designed to allow text to wrap around an element, such as an image. While modern methods like Flexbox and Grid are now more common, understanding floats is essential for legacy code and specific text-wrapping effects.

FLT:LEFT

When an element is floated, it's taken out of the normal document flow and pushed to the left or right of its container. The content that follows will wrap around the floated element, filling the remaining space.

img {
  float: left;
  margin-right: 20px;
}

The Collapse Problem

In the example above, the container might not wrap around the floated element correctly. Because floated elements are out of the normal flow, the parent container can "collapse" to zero height if it contains only floated children.

The Clearfix Solution

The clearfix technique ensures the parent element wraps around its floated children properly. By adding a pseudo-element after the content, we force the container to "see" the floated child.

FIXED

This container uses the clearfix technique. Now, the dashed border correctly encompasses both the image and this text, maintaining the layout flow.

.container::after {
  content: "";  
  display: block;
  clear: both;
}

User Action Pseudo-classes

Provide visual feedback for user interactions without JavaScript.

The Importance of Feedback

User feedback helps users understand the state of interactive elements. CSS provides action pseudo-classes like :hover, :active, :focus, and :visited to automate this feedback.

Interaction Showcase

:hover & :active

Change styles when mousing over or clicking.

button:hover { ... }
button:active { ... }

:focus

Styles when an element gains focus (keyboard/click).

input:focus { ... }

:visited

Styles a link after it has been visited.

Visit Example.com
a:visited { ... }

:checked

Styles checkboxes and radio buttons when selected.

input:checked { ... }

Summary

  • :active – Feedback during a click or activation.
  • :hover – Visual cue for mouse attention.
  • :focus – Critical for accessibility (Tab navigation).
  • :visited – Helps users track their navigation history.
  • :checked – Custom styling for form selections.

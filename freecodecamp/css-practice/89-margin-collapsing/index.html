What is Margin Collapsing?

Margin collapsing occurs when the vertical margins of adjacent elements overlap, resulting in a single margin equal to the larger of the two. This behavior only applies to vertical margins.

20px (bottom) + 30px (top) = 30px (Collapses to Larger)

Prevention Techniques

You can prevent collapsing by introducing a "separator" between margins:

  • Adding a border to the parent.
  • Setting padding on the parent.
  • Using inline-block or flex/grid containers.

Gaps & Spacing

Mastering Rows, Columns, and Tracks

What is a Track?

A track is the space between two neighboring grid lines. In CSS Grid, tracks generally refer to the rows and columns that make up your layout.

To control the space between these tracks, we use the gap properties.

1. Column Gap

Vertical Rhythm

Column Spacing

.container {
  display: grid;
  grid-template-columns: repeat(4, 1fr);
  column-gap: 10px;
}

2. Row Gap

Horizontal Rhythm

Row Spacing

.container {
  display: grid;
  grid-template-columns: 1fr 1fr;
  row-gap: 30px;
}

3. The Gap Shorthand

Combined Power

Simple Shorthand

.container {
  gap: 30px 10px; /* Row Col */
}

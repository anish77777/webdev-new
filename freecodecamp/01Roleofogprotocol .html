Open Graph (OG) is a protocol that allows web pages to control how their content appears when shared on social media platforms.

The og:title tag defines the main heading shown in the social media preview.

The og:type tag describes the type of content, such as website, article, or video.

The og:image tag sets the preview image that appears in social media feeds.

The og:url tag specifies the main URL that platforms should use when the page is shared.

The og:description tag provides a short summary displayed under the title in the preview.

The og:locale tag defines the language and region of the page content.

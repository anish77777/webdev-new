When Is the Alt Attribute Needed, and What Are Some Examples of Good Alt Text?

What is Alternative Text?

Alternative text, often abbreviated as alt text, is a brief text description of an image. It provides essential information about the image for users who cannot see it, such as people who use screen readers and other assistive technologies.

It's essential for making websites accessible to people with visual disabilities.

Alternative text is also used by:

  • Search engines to understand images
  • Browsers which may display it when an image is not loaded properly (due to missing files or connectivity issues)

Examples: Bad vs Good Alt Text

Example 1: Puppy Image

A person with a visual disability will not be able to know what's in the image unless it's described in the alt text.

‚ùå Bad Alt Text

"A cute puppy."

This text is not specific enough to convey the important details of the image:

  • What does the puppy look like?
  • Where is the puppy?
  • Are there any important objects around the puppy?

‚úÖ Good Alt Text

"A black and white puppy with an orange collar lies on its belly in the sand, looking off to the side. A bright orange ball rests near its front paws."

Here's how it looks in HTML:

<img src="puppy.png" alt="A black and white puppy with an orange collar lies on its belly in the sand, looking off to the side. A bright orange ball rests near its front paws." />

Example 2: Resort Image

‚ùå Bad Alt Text

"Resort."

This is too short and doesn't provide enough information about the image.

‚úÖ Good Alt Text

"Tropical resort featuring a swimming pool surrounded by palm trees and bungalows." 
<img src="resort.png" alt="Tropical resort featuring a swimming pool surrounded by palm trees and bungalows." />

Best Practices for Writing Alt Text

  • Keep alt text short: It should be detailed enough to understand the image but not so long that it becomes confusing.
  • Don't describe every little detail: Focus on the most important aspects of the image.
  • Don't start with "image of" or "picture of": Just start the description directly.
  • Avoid redundancy: If there's similar text around the image, you don't need to write it again.
  • End with a period: It's usually recommended to end the alt text with a period for consistency.

Alt Text for Images Used as Links

If the image is a link to another page, instead of describing the image itself, the alt text should describe what will happen if users click on it.

Example: Arrow Icon Link

‚ùå Bad Approach

<a href="about.html">
  <img src="arrow-right.png" alt="Right arrow." />
</a>

This only describes what the image looks like, not its function.

‚úÖ Good Approach

<a href="about.html">
  <img src="arrow-right.png" alt="Go to next page." />
</a>

This describes what will happen when users click on the image.

Decorative Images

Only images that convey important information should have alt text. If an image is only used for decorative purposes, it should have null (empty) alt text, so it can be ignored by screen readers and other assistive technologies.

<img src="decorative_image.jpg" alt="" />
‚ö†Ô∏è Important: Every image on your website should have an alt attribute, even if it's empty. If you omit the alt attribute completely, some screen readers will read the file name instead, which can be distracting for people who use assistive technologies.

Testing Your Alt Text

Before your website is live, you should carefully test if screen readers can read the alt text correctly.

Summary

Writing effective alt text is essential for creating accessible web content. As a web developer, by providing clear descriptions of your images, you can make sure that everyone can engage with your website in an inclusive online experience.

The audio element is used to embed sound content in an HTML document. It allows browsers to play audio files such as mp3, wav, and ogg without requiring external plugins.

The src attribute specifies the location of the audio file. If the audio element does not include the controls attribute, no audio player will be visible on the page.

The controls attribute is a boolean attribute that displays built-in audio controls such as play, pause, and volume.

The loop attribute makes the audio replay automatically after it finishes playing.

The muted attribute starts the audio in a muted state until the user enables sound.

Different browsers support different audio formats. To handle this, you can use multiple source elements inside the audio element, and the browser will play the first format it supports.

The video element is used to embed video content in an HTML document. It supports formats such as mp4, ogg, and webm and can display both video and audio.

The src attribute defines the video file location, while controls enables playback controls like play, pause, and volume.

The loop attribute makes the video replay continuously after it ends.

The muted attribute starts the video with sound turned off. This is commonly required for autoplay to work in modern browsers.

The autoplay attribute starts playing the video automatically when the page loads.

The poster attribute specifies an image that is shown before the video starts playing or while it is loading. This attribute is unique to the video element.

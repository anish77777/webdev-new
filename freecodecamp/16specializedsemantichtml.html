Working with Specialized Semantic Elements

How Do You Display Mathematical Equations and Chemical Formulas in HTML?

The superscript element is used to display text above the normal line. It is commonly used for mathematical equations.

22 = 4

103 = 1000

The subscript element is used to display text below the normal line. It is commonly used for chemical formulas.

Water: H2O

Carbon Dioxide: CO2

How Do You Represent Computer Code in HTML?

The code element is used to represent short inline code snippets.

To set the text color to blue in CSS, use: color: blue;

For multiple lines of code, the code element is placed inside the preformatted text element. 


body {
  color: red;
}

The pre element preserves spacing and line breaks exactly as written.

What Are the U, S, and Ruby Elements Used For, and How Do They Work?

The u Element

The u element represents inline text with non-textual annotation, such as spelling errors.

inccccort spling issses

The s Element

The s element represents text that is no longer accurate or relevant.

Tomorrow's hike will be meeting at noon.

The hike has been canceled.

The ruby Element

The ruby element is used to show pronunciation or annotation, mainly in East Asian typography.

明日 ( Ashita )

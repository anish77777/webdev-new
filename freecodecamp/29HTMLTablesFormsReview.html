üìù HTML Form Elements and Attributes

<form> Element

Purpose: Used to create an HTML form for user input.
Key Attributes:
  • action - Specifies the URL where the form data should be sent
  • method - Specifies the HTTP method to use when sending form data (GET or POST)
<!-- Basic form structure -->
<form method="value-goes-here" action="url-goes-here">
  <!-- inputs go inside here -->
</form>

<input> Element

Purpose: Used to create an input field for user input.
Common Attributes:
  • type - Specifies the type of input (text, email, number, radio, checkbox, etc.)
  • placeholder - Shows a hint to the user about what to enter
  • value - Specifies the value of the input
  • name - Assigns a name to the input field (used as key when form data is submitted)
  • size - Defines the number of visible characters
  • min - Specifies the minimum value (for number inputs)
  • max - Specifies the maximum value (for number inputs)
  • minlength - Minimum number of characters required
  • maxlength - Maximum number of characters allowed
  • required - Input must be filled out before submitting
  • disabled - Input field is disabled
  • readonly - Input field is read-only
<!-- Text input example -->
<input 
  type="text"
  id="name"
  name="name"
  placeholder="e.g. Quincy Larson" 
  size="20"
  minlength="5"
  maxlength="30"
  required
/>

<!-- Number input example -->
<input 
  type="number"
  id="quantity"
  name="quantity"
  min="2"
  max="10"
  disabled
/>

<!-- Button input example -->
<input type="button" value="Show Alert" />

<label> Element

Purpose: Used to create a label for an input field.
Key Attribute:
  • for - Specifies which input field the label is for

Method 1: Implicit Form Association

Inputs can be associated with labels by wrapping the input inside the label element:

<form action="">
  <label>
    Full Name:
    <input type="text" />
  </label>
</form>

Method 2: Explicit Form Association

Inputs can be associated with labels using the for attribute:

<form action="">
  <label for="email">Email Address: </label>
  <input type="email" id="email" />
</form>

<button> Element

Purpose: Used to create a clickable button.
Type Attribute Values:
  • button - Regular button (no default behavior)
  • submit - Submits the form
  • reset - Resets the form to default values
<button type="button">Show Form</button>
<button type="submit">Submit Form</button>
<button type="reset">Reset Form</button>

<fieldset> and <legend> Elements

Purpose:
  • <fieldset> - Groups related inputs together
  • <legend> - Adds a caption to describe the group
<!-- Radio button group -->
<fieldset>
  <legend>Was this your first time at our hotel?</legend>

  <label for="yes-option">Yes</label>
  <input id="yes-option" type="radio" name="hotel-stay" value="yes" />

  <label for="no-option">No</label>
  <input id="no-option" type="radio" name="hotel-stay" value="no" />
</fieldset>

<!-- Checkbox group -->
<fieldset>
  <legend>
    Why did you choose to stay at our hotel? (Check all that apply)
  </legend>

  <label for="location">Location</label>
  <input type="checkbox" id="location" name="location" value="location" />

  <label for="price">Price</label>
  <input type="checkbox" id="price" name="price" value="price" />
</fieldset>

üìä HTML Table Elements and Attributes

Table Elements Overview

Key Table Elements:
  • <table> - Creates an HTML table
  • <caption> - Adds a title to the table
  • <thead> - Groups the header content
  • <tbody> - Groups the body content
  • <tfoot> - Groups the footer content
  • <tr> - Creates a table row
  • <th> - Creates a header cell
  • <td> - Creates a data cell
Important Attribute:
  • colspan - Specifies the number of columns a cell should span 
<table>
  <caption>Exam Grades</caption>

  <thead>
    <tr>
      <th>Last Name</th>
      <th>First Name</th>
      <th>Grade</th>
    </tr>
  </thead>

  <tbody>
    <tr>
      <td>Davis</td>
      <td>Alex</td>
      <td>54</td>
    </tr>

    <tr>
      <td>Doe</td>
      <td>Samantha</td>
      <td>92</td>
    </tr>

    <tr>
      <td>Rodriguez</td>
      <td>Marcus</td>
      <td>88</td>
    </tr>
  </tbody>

  <tfoot>
    <tr>
      <td colspan="2">Average Grade</td>
      <td>78</td>
    </tr>
  </tfoot>
</table>

üõ†Ô∏è Working with HTML Tools

1. HTML Validator

Definition: A tool that checks the syntax of HTML code to ensure it is valid.

Purpose: Identifies errors and warnings in your HTML code, ensuring proper structure and compliance with web standards.

Popular validators:

  • W3C Markup Validation Service (validator.w3.org)
  • JSONFormatter.org HTML Validator

2. DOM Inspector

Definition: A tool that allows you to inspect and modify the HTML structure of a web page.

Purpose: View the Document Object Model (DOM) tree structure and see how elements are organized.

How to access: Right-click on any element and select "Inspect"

3. DevTools (Developer Tools)

Definition: A set of web developer tools built directly into the browser.

Purpose: Helps you debug, profile, and analyze web pages.

Key features:

  • Elements tab - Inspect HTML and CSS
  • Console tab - View errors and warnings
  • Network tab - Monitor network requests
  • Sources tab - Debug JavaScript

How to open:

  • Windows/Linux: Ctrl + Shift + I
  • Mac: Cmd + Option + I
  • Right-click ‚Üí Inspect

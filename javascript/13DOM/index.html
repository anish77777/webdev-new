JavaScript DOM (Document Object Model)

What is DOM?

The DOM (Document Object Model) is a programming interface that represents an HTML document as a tree structure. It allows JavaScript to interact with and manipulate HTML elements and CSS properties dynamically.

Key Points:

  • Tree Structure: HTML documents are represented as a tree of objects
  • Dynamic Manipulation: Change content, styles, and attributes in real-time
  • Event Handling: Respond to user interactions (clicks, typing, etc.)
  • Real-time Updates: Updates appear immediately in the browser without page reload

Common DOM Operations:

  • Select elements: getElementById(), querySelector(), getElementsByClassName()
  • Change content: textContent, innerHTML
  • Change styles: style property
  • Add/Remove classes: classList.add(), classList.remove()
  • Create elements: createElement()
  • Add event listeners: addEventListener()
